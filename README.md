# ЛР35 на Vue

Макет онлайн-магазина с навигацией и редактированием записей с помощью vue-router и bootstrap

## Установка

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
